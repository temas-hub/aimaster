2. раз в кадр 
	посылаем все команды в очереди
	обновляем положение мешени с учетом dt=delta, 
		x,y =  += v * dir * dt
	для всех снарядов
		еще летит
			x,y =  += v * dir * dt
		остановился
			x,y =  += target.v * target.dir * dt
2. Как только принимаем обновление от сервера
	if (timestamp > lasttimestamp)
		чистим из очереди все команды до последнего timestamp
		обновляем модель мишени
			dir = server.dir
			v = server.v
		обновляем модель снарядов
			еще летит
				dir = server.dir
				v  = server.v
			остановился
				v  = server.v = 0
				x=server.x
				y=server.y
сервер

	создает снаряд (нач. точка, скорость(v), ускорение трения(a), направление)
	обвновляет ему данные с учетом dt=(тек. время - время создания)		
		x,y =  += v*dt
		v  -= a*dt
2. Каждый тик		
	Для каждого снаряда (Box2D)
		2.1 обновляется dt = дельта обновления сервера
			x,y =  += v*dt
			v  -= a*dt
		2.2. Считается пересечения с объектами
			2.2.1 - стенка, откражне -> скорость, направление
			2.2.2 - другой снаряд -> скорость, направление
